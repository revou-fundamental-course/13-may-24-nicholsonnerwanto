Luas Segitiga

Rumus Luas Segitiga, yaitu :

L = 1/2 x a x t

Dimana :
L = Luas
a = Alas
t = Tinggi

Hitung Luas Segitiga

Keliling Segitiga

Rumus Keliling Segitiga, yaitu :

K = S1 (AB) x S2 (BC) x S3 (CA)

Dimana :
K = Keliling
S1 / AB = Sisi 1
S2 / BD = Sisi 2
S3 / CA = Sisi 3
